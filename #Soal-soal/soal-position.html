position: absolute

Properti untuk mengatur posisi suatu elemen. resource
Berikut contohnya :

absolute: Tetap, relatif terhadap parent
absolute: Tetap, relatif terhadap parent
relative: relatif terhadap tempat awal diposisikan
relative: relatif terhadap tempat awal diposisikan
sticky: awalnya relatif jadi fixed
static: mirip relatif
fixed: relatif terhadap viewport
























Sifat-sifat:

Umum

  • Untuk mengatur posisi menggunakan ( top,right,bottom,left )
    Contoh: position: relative; top: 30px; left: 100px;
  • Properti tidak boleh kontradiksi || left != right & bottom != top
    Contoh:
    top: 10px; bottom: 10px
    atau
    left: 10px; right: 10px

Absolute

  • Posisi tetap, dalam artian tidak dapat "tergeser" / "terdorong" oleh elemen lain (bisa numpuk) *lihat contoh
  • Tidak dapat menggunakan margin
  • Posisi relatif terhadap parent
  • Parent harus memiliki posisi resource
  • Dapat digunakan untuk meng-center-kan <div>

Fixed

  • relatif terhadap viewport (POV user)

Static

  • properti ( top,right,bottom,left ) tidak dapat bekerja

Sticky

  • Awalnya relative kalo udh nyampe posisi scrollnya (relatif parent bukan viewport) bakal jadi fixed
  • Jarang dipake (opinion)
